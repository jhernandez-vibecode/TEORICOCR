Cómo anular un PIN

  1. 1. Abrí el Google Sheet "CONTROL DE TOKENS"
  2. 2. Buscá el PIN en la columna A
  3. 3. Cambiá la columna F (ESTADO) de ACTIVO a:
  4. → ANULADO para bloqueo permanente
  5. → SUSPENDIDO para bloqueo temporal (reactivable)
  6. 4. El bloqueo es instantáneo, sin redeploy
  7. 5. Para reactivar: volvé a escribir ACTIVO
¿Cómo configurar Google Apps Script?

Hacé esto una sola vez:

  1. 1. Abrí tu Google Sheet "CONTROL DE TOKENS"
  2. 2. Menú → Extensiones → Apps Script
  3. 3. Borrá todo el código que aparece y pegá el código del archivo apps-script.gs que te entregamos
  4. 4. Guardá (Ctrl+S) y dale un nombre al proyecto
  5. 5. Clic en Implementar → Nueva implementación
  6. 6. Tipo: Aplicación web
  7. 7. Ejecutar como: Yo
  8. 8. Quién tiene acceso: Cualquier usuario
  9. 9. Clic en Implementar → autorizá los permisos
  10. 10. Copiá la URL de la aplicación web y pegala arriba en "Configuración"

⚠️ Si modificás el script después, debés hacer una nueva implementación (no editar la existente) para que los cambios tomen efecto.
